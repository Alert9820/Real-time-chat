About The App

This chat application was developed as a project to demonstrate modern web development techniques and real-time communication capabilities.

Key Features:

  • Real-time messaging
  • Voice and video calling
  • Group chats
  • End-to-end encryption
  • Cloud synchronization
  • Cross-platform compatibility

The app is built with cutting-edge technologies including Node.js, Socket.io, and MongoDB, ensuring a seamless and responsive user experience.
